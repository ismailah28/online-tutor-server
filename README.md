# online-tutor-api

> Backend Server for an online tutor app

## Usage

Create a .env file in your root directory and add the follwing along with their values:

```
- MONGO_URL=
- JWT_SECRET=
- JWT_EXPIRE=
- PORT=
```

## Install Dependencies

```
yarn
```

## Run App

```
# Run in dev mode
yarn run dev

# Run in prod mode
yarn start
```
